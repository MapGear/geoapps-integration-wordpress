Embed a GeoApps map with ease in your Wordpress website

== Description ==

GeoApps is a modern apps platform, focused on location-oriented information. With this wordpress plugin, you can add a GeoApps map
in your Wordpress-based website without any coding. The only requirement is an active GeoApps subscription and the id of the configured map.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/geoapps` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the block in your page (only available as Gutenberg block)

== Frequently Asked Questions ==

= I don't have a GeoApps subscription, how can I use this plugin?

You can purchase a GeoApps subscription through www.geoapps.nl

= I am trying to configure this block, but it isn;t working?

You need to verify that the map that you are trying to link is configured as a public map. Maps that are
private are currently not supported in this plugin.

= I am missing some features

Please let us know which features you are missing by using this plugin. You can provide us with feedback by leaving a comment,
add the requested feature on feedback.geoapps.nl or send an email to support@geoapps.nl.
